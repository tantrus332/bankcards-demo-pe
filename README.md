
# Bank Cards Management System

Простой сервис для управления банковскими картами с поддержкой шифрования данных (AES), ролевой моделью доступа.

## Технологический стек

* **Java 17** (Spring Boot 4.x)
* **PostgreSQL** (хранение данных)
* **Spring Data JPA / Hibernate**
* **Spring Security & JWT** (авторизация)
* **Docker & Docker Compose** (контейнеризация)
* **Lombok** (генерация кода)

---

## Инструкция по запуску

### 1. Требования
* Установленный Docker и Docker Compose.
* (Опционально) Java 17+ и Maven, если запуск планируется без Docker.

### 2. Настройка переменных окружения
Переименовать файл .env.example в .env

В файле (.env) настроить:
- Логин, пароль, адрес и порт СУБД
- Настроить секретный JWT ключ
- Настроить длительность действия JWT
- Указать ключ шифрования номеров карт (длина ключа обязательно должна быть: 16, 24 или 32 символа)

### 3. Запуск через Docker

Приложение и база данных поднимутся автоматически:

```bash
# Сборка и запуск (выполнить в папке с файлом docker-compose.yml)
docker-compose up --build

```

После завершения сборки API будет доступно по адресу: `http://localhost:8080/api/v1`
